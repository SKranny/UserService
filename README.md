# Java Spring project "Social networking site"
### User-service
## Description
User service is responsible for all functionality related to users of our social network, except authorization. For example: 
- user registration
- deleting user accounts
- retrieving account information and many others.
and many others.
## Technologies
- Java version 11
- Spring Framework
- Flyway
- Lombok
- Mapstruct
- Spring Data JPA
- PostgreSQL
- Spring Security
- Spring Cloud OpenFeign
- Spring Cloud Netflix Eureka
- JWT(JsonWebToken)
- Nexus repository
- Kafka
- Swagger OpenApi
